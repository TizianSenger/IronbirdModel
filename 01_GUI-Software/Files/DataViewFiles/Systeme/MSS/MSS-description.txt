Das Mission Support Subsystem dient der Planung von Missionen für das Waffensystem Eurofighter. Die Piloten können mit dieser Software unter anderem ihren Flugplan erstellen, Wegpunkte und Missionsziele definieren, Radiofrequenzen festlegen, den Kraftstoffverbrauch berechnen. Die fertige Missionsdatei wird auf einen Portable Data Store geschrieben und vom Piloten im Flugzeug einspielt.
